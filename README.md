## Como executar o projeto
### Back-end
```bash
# Vá para a pasta backend
$ cd backend

# Duplique o arquivo .env
$ cp .env.example .env

# Altere a variável de ambiente
$ nano .env

# Instale as dependências
$ npm install

# Execute a aplicação
$ npm start
```

### Front-end
```bash
# Vá para a pasta frontend
$ cd frontend

# Duplique o arquivo .env
$ cp .env.example .env

# Altere a variável de ambiente
$ nano .env

# Instale as dependências
$ npm install

# Execute a aplicação
$ npm start
```
